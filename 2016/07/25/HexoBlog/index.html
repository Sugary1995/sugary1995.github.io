【HEXO】PERSONAL BLOG

2016-07-25

【Information】

  • Name：The Blog Of Sugary
  • Development environment：
    • OS：window
    • Blog framework：Hexo3

【Steps】

Install prerequisite software

  • Node.js：https://nodejs.org/en/
  • Git Client：http://git-scm.com/download

Install hexo

  • 输入安装Hexo的命令：
  • npm install -g hexo
  • 【注意】如果安装太慢，可以修改配置
  • npm config set registry http://registry.cnpmjs.org
  • npm info underscore
  • 安装完成后，在喜欢的文件下，执行以下指令，Hexo即会自动在目标文件夹建立网站所需要的所有文章。
  • hexo init
  • 安装依赖包：
  • npm install
  • 在E:\hexo内执行以下命令：
  • hexo g
  • hexo s
  • 然后用浏览器访问http://localhost:4000， 此时，你应该看到了一个漂亮的博客了，当然这个博客只是在本地的，别人是看不到的，hexo3.0使用的默认主题是landscape。
  • 【注意】如果输入地址后一直无响应，可能端口号4000被占用
  • 输入命令，修改端口号： hexo server -p 5000
  • 用浏览器访问http://localhost:5000
